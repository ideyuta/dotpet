# dotpet

ターミナルに住むペット。tmux のステータスバーでアニメーションし、放置するだけで冒険してアイテムを集める。

## 概要

- tmux ステータスバーに常駐するASCIIペット
- 時間経過で自動的にXPを獲得してレベルアップ
- ランダムイベントでバトル・アイテム発見が発生
- 5.5億通りのプロシージャル生成アイテム
- 装備するとペットのAAに武器・防具がオーバーレイ表示
- Lv.20で転生、次世代へ

```
⣾⣷ Lv.1 木の笛
```

## セットアップ

```bash
go build -o ~/.local/bin/dotpet .
```

### tmux ステータスバー

```tmux
set -g status-right "#(~/.local/bin/dotpet)"
```

### tmux ポップアップ（詳細画面）

```tmux
bind P display-popup -w 58 -h 80% -E "~/.local/bin/dotpet watch"
```

## 使い方

```bash
dotpet          # ステータスライン（1行、tmux用）
dotpet status   # 詳細表示
dotpet watch    # インタラクティブモード
dotpet log      # 冒険ログ
```

### watch モード操作

| キー | 動作 |
|------|------|
| `2` / `i` | もちもの一覧 |
| `1` / `Esc` | ステータスに戻る |
| `j` / `k` | スクロール |
| `q` | 終了 |

## ゲームシステム

### レベルと経験値

1分経過ごとに1XP。Lv.1→2は15XP、Lv.19→20は435XP。全体で約3日でLv.20到達。

### 冒険イベント

10分ごとに判定。バトル（25%）、アイテム発見（10%）、特殊イベント（10%）、何もなし（55%）。

### アイテム

`[接頭辞][素材]の[種類][接尾辞]〈属性〉` の形式で生成。

- 素材 65種（木、鉄、ミスリル、竜骨、混沌の核...）
- 種類 40種（剣、杖、鎧、指輪...）
- 接頭辞 45種 + なし
- 接尾辞 40種 + なし
- 属性 15種 + なし

レア度は6段階: ★ふつう → ★★上質 → ★★★希少 → ★★★★秘宝 → ★★★★★伝説 → ★★★★★★神話

### 種族

cat / dog / bird / frog（ユーザー名のハッシュで決定）

### 転生

Lv.20到達で自動転生。次世代は遺産ボーナス（+1 つよさ）付きで Lv.1 から。

## データ

```
~/.config/dotpet/pet.json
```
